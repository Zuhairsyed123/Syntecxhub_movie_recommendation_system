# Movie Recommendation System using Machine Learning

A content-based movie recommender built with **Python**, **scikit-learn**, and **cosine similarity**.

## Features
- Content-based filtering using genres, keywords, cast, crew, and overview
- Streamlit web interface
- Fast recommendations
- Easy to extend (hybrid, collaborative filtering, deep learning)

## Dataset
Download `tmdb_5000_movies.csv` from:
[TMDB 5000 Movie Dataset on Kaggle](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)
